# Message Emotion Detection

## Project Details
This project aims to detect emotions from messages. 

## Dataset
* Name:- ISEAR Dataset (https://www.kaggle.com/shrivastava/isears-dataset)
* Instances:- 7516
* Columns:- label, text
* Emotions(label):- Anger, Disgust, Fear, Guilt, Joy, Sadness, Shame
